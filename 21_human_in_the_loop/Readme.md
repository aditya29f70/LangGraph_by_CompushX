## What is HITL

- HITL (**human-in-the-loop**) is a design approach in ai system where a human actively participate at critical points of the ai workflow - either to **supervise**, **approve**, **currect**, or **guide** the model's output

- think of hitl as putting a human 'checkpoint' inside an AI pipeline so that important decisions are not mode autonomously by the model

## why hitl exists?

1. to help agentic systems
2. to add accountability

## HITL ensure

- Accuracy
- Safety
- Ethical alignment
- Better user experience

## common hitp patterns:

- action approval pattern (approve/ reject before execution)
- output review /edit pattern
- ambiguity clarification pattern
- Escalration pattern
